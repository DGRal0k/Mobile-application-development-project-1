# BudgetFlow - Mobile Expense Manager

BudgetFlow is an Android-based utility application designed to help users track their spending in real-time. This project was developed as part of the Mobile Application Development curriculum.

## Technical Features
- **UI:** XML-based Material Design.
- **Backend:** Java with Android Lifecycle management.
- **Database:** SQLite for local data persistence.
- **Architecture:** Model-View-Controller (MVC).

## How to Run
1. Clone the repository.
2. Open in Android Studio.
3. Build and run on an Emulator or physical Android device.
